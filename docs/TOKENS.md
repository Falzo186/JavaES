# Catálogo de Tokens (JAVAES)

- IDENTIFICADOR: nombres de variables y métodos
- ENTERO: literales enteros (ej. 123)
- DECIMAL: literales decimales (ej. 3.14)
- CADENA: literales de cadena entre comillas dobles
- BOOLEANO_LIT: valores `verdadero` / `falso`

Palabras reservadas:
- entero, decimal, cadena, booleano, vacio
- si, sino, mientras, retornar, clase, publico, privado

Operadores y delimitadores se nombran en español: `MAS`, `MENOS`, `POR`, `DIV`, `IGUAL`, `DIF`, `MAYOR`, `MENOR`, `ASIG`, `PUNTO_COMA`, etc.

Errores léxicos se reportan con ubicación (línea/columna).
